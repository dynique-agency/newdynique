# Dynique - Ultra Premium Website

Een ultra premium website met Next.js, featuring een elegante header, hero sectie met video achtergrond, en premium content secties in puur zwart en wit.

## Features

- Responsive design met mobile menu
- Video achtergrond in header en hero sectie
- Smooth scroll animaties
- Premium black & white design
- Modern UI/UX met hover effecten
- TypeScript + Tailwind CSS

## Getting Started

Installeer dependencies:

```bash
npm install
```

Start de development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser.

## Project Structuur

- `/src/app` - Next.js App Router paginas
- `/rsc` - Resources (video bestanden)
- `/public` - Statische bestanden

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Video background integratie
